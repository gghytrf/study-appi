# あマ指・はり師きゅう師 国試対策アプリ

あん摩マッサージ指圧師・はり師きゅう師の国家試験対策学習アプリです。
ブラウザだけで動作し、学習履歴は各端末のローカル（IndexedDB）に保存されます。

## 利用方法

下記URLにアクセスするだけで利用できます。インストール不要。

**📱 アプリURL：** `https://<GitHubユーザー名>.github.io/<リポジトリ名>/`

スマホ・タブレット・PCのブラウザで動作します。
ホーム画面に追加すれば、アプリのように起動できます（PWA対応）。

## ファイル構成

```
.
├── index.html              # アプリ本体（HTML/CSS/JS全部入り）
├── data/
│   └── questions.json      # 問題データ（編集はここだけ）
├── docs/                   # 各種マニュアル・設計書
│   ├── GitHub_Desktop_マニュアル.md
│   ├── 設計書.md
│   └── 運用ガイド.md
├── README.md               # このファイル
└── .nojekyll               # GitHub Pagesのビルドを無効化（必須）
```

## 学習履歴について

- **保存場所**：各端末のブラウザ内（IndexedDB）。サーバには送信されません
- **デバイス間共有**：されません（A端末の進捗はB端末には反映されない）
- **バックアップ**：アプリ内の「📥 出力」ボタンでJSONファイルを保存できます
- **5日ごとの通知**：最終バックアップから5日以上経過すると、起動時にバナーで案内されます
- **復元**：「📤 取込」ボタンから保存したJSONファイルを選択

### iPhone Safariの注意点

iOSの仕様で、**7日間アプリを開かない**とブラウザ内データが自動削除される可能性があります。
週1回以上は起動するか、定期的にバックアップを取ってください。

## 更新作業（管理者向け）

問題データの追加・修正は `data/questions.json` を編集するだけです。
詳しくは `docs/GitHub_Desktop_マニュアル.md` を参照してください。

問題データのバージョン（`questionDataVersion`）を更新すると、
次回起動時にすべての端末で自動的に新しい問題データに更新されます。

## ライセンス・利用について

本アプリの問題データは学習目的で作成された私的教材です。
利用は管理者から招待された方に限定されます。
